# PRD ·《心声调频 InnerTune：AI 个性化自我对话声景工具》

**副标题**：用自己的声音，生成一段把自己带回来的音乐。
**赛事**：Build For Music+ 腾讯音乐 AI Hackathon 2026 ｜ **初稿截止 2026-06-25**
**版本**：v0.2（情绪转化闭环）｜ 状态：闭环已跑通，真实 DeepSeek 已接入

---

## 1. 一句话
输入当下状态 → AI 理解情绪并给出生成策略 → 生成第一人称肯定语与核心心声 → 用本人声音录制 →
选声景配方混成专属音频 + 封面 → **记录聆听前后情绪变化** → 保存 / 隐私安全分享。

## 2. 目标用户与场景
面试 / 考试前紧张、睡前内耗、自我怀疑、想坚持目标的年轻人。
主打单场景：**面试 / 考试前的 90 秒自信声景**（情绪强、风险低、Demo 易懂、结果可衡量）。

## 3. 痛点 → 价值
- 情绪混乱时组织不出积极语言 → AI 把烦恼转成贴合处境的自我对话。
- 文字鼓励容易被忽略 → 用**自己的声音**录制核心心声，陪伴感最强。
- 普通 AI 音乐很卷 → 生成的不是一首歌，而是「和自己对话的声音仪式」，且**可量化情绪变化**。

## 4. 核心闭环（P0，已实现 · 6 步向导）
选场景 → 三段输入（现在状态 / 想抵达 / 不想听到）→ AI 理解+策略卡片 → 肯定语+核心心声（现在时·正面·无否定词，可编辑）→
本人录音（核心心声为主 + 授权勾选）→ **选背景音**（AI 配乐+赫兹 / 上传本地 / QQ音乐占位 / 不添加）→
**调参混音台（潜听模式）**：背景音(音量) · 人声(速度1–2x / 音量 / 叠加音轨0–3 / 音轨交错 / 自动循环铺满) · 合成(总时长≤30min / 双耳节拍 / 8D) → 四阶段生成 →
**情绪前后评分（紧张/自信 1–5，展示变化）** → 下载 / 隐私分享 / 保存到「我的音轨」。

> 录音后提供「⚡ 用推荐配乐直接生成」快捷路径，跳过背景音/调参两步，让现场 demo 仍然快。
> 潜听模式：人声以极低音量循环铺在背景音之下，回归「自我暗示」的原始设计。

## 5. 核心创新
| 创新 | 说明 |
| --- | --- |
| 自我声音 | 用户听到的是自己对自己说（具身化） |
| 情绪转译 | 从负面叙述转为积极自我对话，并展示 AI 的理解与策略 |
| AI 声景 | 不只是文字，而是程序化合成、零版权的音乐化体验 |
| 可反馈 | 聆听前后情绪评分构成闭环（产品而非玩具） |
| 可传播 | 隐私安全的声景卡片，不泄露原始倾诉 |

## 6. 成功标准
- 现场 90 秒内完成一条 case 的「输入 → 生成 → 录音 → 混音 → 播放 → 情绪评分」。
- 三个固定 demo case 稳定复现。
- 风险可控：反玄学、反医疗、AI 标识、危机词安全兜底、本人授权。

## 7. 技术架构
状态输入 → LLM 结构化 JSON → 本人录音 → 背景音来源（程序化配方 / 上传解码）+ 逐轨混音参数 →
Web Audio `OfflineAudioContext` 离线混音（背景音 ducking、人声变速/循环、音调、双耳节拍、8D 自动声像、限幅归一化）→
封面 → 播放/评分/分享。纯前端音频管线；唯一服务端是一次 DeepSeek 调用（含本地模板兜底，无 key 也能 demo）。

## 8. 非目标（本期不做）
完整声音克隆、整首歌生成、上传流行歌做 BGM、社区 Feed、心理测评、玄学打卡。

## 9. Roadmap
P1：分享卡片优化、波形可视化、TTS 兜底、更多声景配方。
P2：情绪声景日记、每日打卡、私人心声歌单、与 K 歌/歌单/睡眠音频联动、合规本人音色定制。
